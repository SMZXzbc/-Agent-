# 🎓 多Agent协作智能学术助教系统

基于多Agent协作的智能答疑系统，通过长链推理与共享工作记忆实现高质量学术辅导。

## 🏗️ 系统架构

| Agent | 职责 | 核心技术 |
|-------|------|---------|
| **意图解析Agent** | 分析学生问题的深层需求 | 长链推理、概念提取 |
| **知识检索Agent** | 在课程知识库中检索相关内容 | RAG检索、关联扩展 |
| **生成校验Agent** | 生成回答并校验事实一致性 | 事实校验、难度适配 |
